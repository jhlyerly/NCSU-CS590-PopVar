CS 590 Fall 2025 - Lab 7 - Using PopVar to predict crosses

This is a ReadMe file with instructions for how to get started with this example project for Lab 7.

R and RStudio should already be installed on your computer. 
To install R and RStudio:
https://rstudio-education.github.io/hopr/starting.html

Download the folder from google drive to the location of your choice on your computer. The downloaded folder should contain the R project, the markdown file, and subfolders for input files and output files.

Open the R project file CS590_Lab7_PopVar. Then open the markdown file CS590_Lab7_PopVar.

You may be prompted to install any libraries that you do not have installed already for this exercise.
